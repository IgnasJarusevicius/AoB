# AoB

2D Game university project

Graphics assets taken from [Reiner's Tilesets](https://www.reinerstilesets.de/)

## Dependencies

* OpenGL 
* Freeglut
* GLEW
* OpenAL

## Installation

#### Linux

<pre>
sudo apt install libglu1-mesa-dev freeglut3-dev libglew-dev libopenal-dev
mkdir build
cd build
cmake ..
make
</pre>
